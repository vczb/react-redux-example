## Instalação

- git clone https://github.com/vczb/react-redux-example.git
- cd react-redux-exampe
- npm install
- npm start
- json-server server.json -p 3333 ( Em outro terminal )
- http://localhost:3000/

## Tecnologias

- React
- Axios
- History
- Immer
- Redux
